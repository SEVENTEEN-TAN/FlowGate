# FlowGate (流驭)

**FlowGate** 是一个将 **「脚本工作流编排」** 与 **「客户端UI动态下发」** 和 **「卡密授权鉴权」** 完美结合的轻量级无服务器调度引擎。

它不仅能让你在浏览器中通过连线的方式编排 Python / JavaScript 代码，还能将这些脚本包装成拥有精美独立 UI 的应用程序，并通过“卡密库”进行分发和商业化变现。

## 🌟 核心特性

- 🛠 **可视化工作流编排 (Workflow Editor)**
  - 基于 React Flow 构建的 DAG（有向无环图）执行引擎。
  - 原生支持自定义 **Python 子进程** 和 **JavaScript 沙箱(VM)** 脚本节点的无缝混编。
  - 支持参数上下文的提取与上下游插值替换，支持多分支判断。

- 🎨 **动态 UI 设计器 (Dynamic UI Builder)**
  - 拖拽式表单搭建（文本、数字、下拉框、富文本展示、开关等）。
  - 管理端设计的界面会动态分发并渲染给最终用户使用。
  - 每一个按钮都可以绑定后端设计好的独立工作流逻辑。

- 🔑 **卡密授权与额度管理 (Access Control & Key Pools)**
  - **灵活的发卡模式**：支持按次（成功后失效）、时间段（月卡/周卡）、不限制等多种核销逻辑。
  - **卡密库**：管理员可以将不同类型的应用权限归类到独立卡密库，精准控制发卡额度。
  - 用户只需输入有效的口令，即可登入对应的应用并在沙箱中执行您定义的脚本。

- 📊 **数据看板与实时观测 (Operations Dashboard)**
  - 实时统计平台使用应用总数、制卡总数、工作流总数以及累计执行流水。
  - 自动呈现近 30 天接口调用的成功/失败 **流量动态柱状图**。
  - 完善的卡密池 **库存预警**（当特定类别可用发卡量不足时进行亮红报警）。
  - **实时日志展示**：用户端在执行工作流时如同云云函数一样实时看到每一步算子吐出的日志。

## 🚀 快速开始

本项目前后端同构，开发模式下使用 `vite` 代理全栈及热重载，后端使用 `express` + `better-sqlite3`。

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器 (前后端将同时启动)
npm run dev
```

启动后，访问 `http://localhost:3000/admin` 进入管理后台。
- **首次进入**会自动触发系统初始化设置，创建初始管理员账号密码。
- 后台可以管理工作流、页面设计、发卡配置。

## 🛠 技术栈

- **Frontend**: React 18, React Router v6, Tailwind CSS, Lucide Icons, React Flow (节点编辑器)
- **Backend**: Express, Better-SQLite3, Node `:vm` module, JWT Auth
- **Build**: Vite + tsx
